First Introduction to Programming.

10th Grade programming was my first introduction to programming and programming software. During this year, we learned basic HTML and CSS and made all of our projects on Replit. What we learned:

In 10th Grade we learned basic html, css, and how information travels across the internet. Information on the internet travels as digital data, known as bits which is broken down into small packets and transmitted over networks using protocols like TCP/IP.
