What we learned

Our main objective throughout this school year was memorizing the syntaxes in order to be well prepared for the WebXam. We did this by learning more advanced html and css as well as a bit of javascript. We learned how a simple code is structured and the three types of css, which are, inline, internal, and external. I learned that html is backbone of a code, laying the foundation for a dynamic webpage. Furthermore, it's the standard markup language used to create and structure content on the internet. CSS is design aspect of programming, it lets you customize and decorate you webpages to make them more appealing. Lastly, Javascript is what gives the webpage functionality through functions, these are possible to do in buttons or text.
